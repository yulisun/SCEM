# SCEM
## 相关算法

基于GMMSP的超像素分割方法下载自Dr. Ban Zhihua的主页：https://github.com/ahban/GMMSP-superpixel ，如果使用该方法，请引用其相关论文。

## 代码运行

解压Zip文件(GMMSP-superpixel-master)，运行SCEM_demo.m。代码已在Matlab 2016a版本下测试。
